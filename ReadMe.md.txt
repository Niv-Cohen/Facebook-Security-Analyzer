Here some instructions in order to run my project.

1.install all the dependencies 
2.Download Origins and Cors changer plugin -make sure you enable it
3.open the entire project in pycharm
4.make sure you flush all previous DB if exists - py py.manage flush
5.Run the command "py manage.py runserver" in the terminal.
6.Register new user with your Account's email,name and password.
7.After Completing this process press New Scan-after the scan Your Security Graph will be displayed
Enjoy :)
